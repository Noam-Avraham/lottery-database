About the Database
Only the modern format — consistent and comparable data

The Israeli Lotto changed its format several times over the years. To ensure all draws in the database use the same rules, only draws from March 2011 onwards are included — when the pool became 1–37 main numbers and 1–7 strong number, and has remained that way ever since.

✕
1999 – March 2004   Excluded
Main pool: 1–49  ·  ~429 draws
✕
March 2004 – Feb 2009   Excluded
Main pool: 1–34  ·  Strong: 1–10  ·  ~549 draws
✕
Feb 2009 – March 2011   Excluded
Main pool: 1–37  ·  Strong: 1–8 (transitional)  ·  ~221 draws
✓
March 2011 → Present   In Database
Main pool: 1–37  ·  Strong: 1–7  ·  1,672 draws

The full database (all 1,672 modern draws including prize tier breakdowns) is available on GitHub .
